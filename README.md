# igtpOracleApi

DataBase for igtp 5.0 Server (Oracle PLSQL)

1. Datos Geograficos
    1. Regiones
    2. Provincias
    3. Municipalidades
    4. Ciudades  
    5. **Locaciones**
2. Datos Comunes
    1. Tipos de Cargas
    2. Tipos de Vehiculos
    3. **Clientes**
    4. Remolques
    5. Tractomulas
3. Datos Logisticos
    1. **Rutas**
    2. **Fletes**
    3. **Transferencias**
4. Datos Facturacion a Clientes
5. Datos Pagos a Conductores
